<p class="tb-section-title">使い方</p>

1. サイトにアクセス
2. メディアURL（動画リンク）を入力
3. 希望する出力フォーマットを選択
4. ダウンロード開始 → ブラウザーに直接保存

<p class="tb-section-title">技術構成</p>

| 項目 | 内容 |
|---|---|
| フレームワーク | React |
| ランタイム | Bun |
| ルーター | React Router |
| ビルドツール | Bun bundler (`Bun.build`) |
| バックエンド連携 | yt-dlp-api |
