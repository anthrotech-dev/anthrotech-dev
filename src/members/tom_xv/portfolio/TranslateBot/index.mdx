<p class="tb-section-title">使い方</p>

1. 翻訳したいメッセージに **国旗リアクション** を付ける
2. Botが翻訳結果をリプライで返信
3. または右クリック → **アプリ → Translate** で翻訳先を選択

<p class="tb-section-title">技術構成</p>

| 項目 | 内容 |
|---|---|
| 言語 | TypeScript |
| ランタイム | Bun v1.3.6+ |
| 翻訳API | DeepL API |
| キャッシュ | Redis |
| Botフレームワーク | discord.js |
| シャーディング | ShardingManager |
