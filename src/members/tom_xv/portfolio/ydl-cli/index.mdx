<p class="tb-section-title">インストール方法</p>

**グローバルインストール（推奨）:**
```bash
npm i -g @tomxv/ydl-cli
```

または

```bash
bun add -g @tomxv/ydl-cli
```

**ローカル開発用:**
```bash
cd ydl-cli
bun link
```

<p class="tb-section-title">使用方法</p>

```bash
ydl https://example.com/video
```

```bash
ydl --help
```

```bash
man ydl
```

<p class="tb-section-title">動作要件</p>

- **Bun v1.0+** - Bun ランタイムが必須です（Node.js のみでは非サポート）
