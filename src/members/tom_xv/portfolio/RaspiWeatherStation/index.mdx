# Raspi Weather Station Board

<div class="tb-hero">
  <p class="tb-subtitle">Raspberry Pi + Chromiumキオスクで動く、案内掲示板風の天気ダッシュボードです。主要都市・長野県の気象情報をリアルタイム表示します。</p>
  <div class="tb-badges">
    <span class="tb-badge ts">TypeScript</span>
    <span class="tb-badge bun">Bun</span>
    <span class="tb-badge raspi">Raspberry Pi</span>
    <span class="tb-badge meteo">Open-Meteo</span>
    <span class="tb-badge jma">気象庁API</span>
  </div>
</div>

<p class="tb-section-title">主な機能</p>

<div class="tb-features">
  <div class="tb-feature">
    <span class="tb-feature-icon">🌤️</span>
    <span class="tb-feature-title">リアルタイム天気表示</span>
    <span class="tb-feature-desc">主要都市と長野県エリアの気象情報をOpen-Meteo APIで取得・表示</span>
  </div>
  <div class="tb-feature">
    <span class="tb-feature-icon">🚨</span>
    <span class="tb-feature-title">警報・注意報</span>
    <span class="tb-feature-desc">気象庁の警報・注意報を生文でリアルタイム表示</span>
  </div>
  <div class="tb-feature">
    <span class="tb-feature-icon">🖥️</span>
    <span class="tb-feature-title">システム監視</span>
    <span class="tb-feature-desc">CPU温度・負荷・メモリ・ディスク・稼働時間・IPアドレスを一覧表示</span>
  </div>
  <div class="tb-feature">
    <span class="tb-feature-icon">🌐</span>
    <span class="tb-feature-title">多言語自動ローテーション</span>
    <span class="tb-feature-desc">日本語・英語・中国語・韓国語を自動で切り替えて表示</span>
  </div>
  <div class="tb-feature">
    <span class="tb-feature-icon">📱</span>
    <span class="tb-feature-title">レスポンシブ対応</span>
    <span class="tb-feature-desc">キオスク画面だけでなくモバイルブラウザでも快適に閲覧可能</span>
  </div>
  <div class="tb-feature">
    <span class="tb-feature-icon">🔄</span>
    <span class="tb-feature-title">自動起動</span>
    <span class="tb-feature-desc">LXDE自動起動設定でRaspberry Pi起動時にキオスクモードで立ち上がる</span>
  </div>
</div>

<p class="tb-section-title">技術構成</p>

| 項目 | 内容 |
|---|---|
| 言語 | TypeScript |
| ランタイム | Bun |
| 天気API | Open-Meteo (CC BY 4.0) |
| 警報データ | 気象庁警報JSON |
| システム情報 | ローカル `/api/pi-status` |
| 動作環境 | Raspberry Pi + Chromiumキオスク |
